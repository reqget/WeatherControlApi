# WeatherControlApi

## Overview

WeatherControlApi is a simple ASP.NET Core Web API project that retrieves weather data from an external API and stores it in a database.

---

## Features

* Fetch weather data from external API
* Store weather information in database
* RESTful API endpoints
* Swagger support for testing

---

## Technologies

* ASP.NET Core Web API
* Entity Framework Core
* MySQL
* HttpClient

---

## Endpoints

* `GET /api/weather` → Get weather data
* `POST /api/weather` → Fetch and save weather data

---

## Run

```bash
dotnet run
```

Swagger:

```text
https://localhost:xxxx/swagger
```

---

## Description

This project demonstrates how to integrate an external API with a .NET backend and persist data using a relational database.
